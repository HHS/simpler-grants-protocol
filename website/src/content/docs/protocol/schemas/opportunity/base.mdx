A funding opportunity, such as a grant or loan.

### Data model

<Tabs>
  <TabItem label="Table">
  | Property       | Type                                                                            | Required | Description                                                  |
  | -------------- | ------------------------------------------------------------------------------- | -------- | ------------------------------------------------------------ |
  | id             | [uuid](/simpler-grants-protocol/protocol/schemas/base-types#uuid)               | Yes      | Globally unique id for the opportunity                       |
  | title          | [string](/simpler-grants-protocol/protocol/schemas/base-types#string)           | Yes      | Title or name of the funding opportunity                     |
  | status         | [OppStatus](/simpler-grants-protocol/protocol/schemas/opp-status)               | Yes      | Status of the opportunity                                    |
  | description    | [string](/simpler-grants-protocol/protocol/schemas/base-types#string)           | Yes      | Description of the opportunity's purpose and scope           |
  | fundingDetails | [OppFunding](/simpler-grants-protocol/protocol/schemas/opp-funding)             | Yes      | Details about the funding available                          |
  | keyDates       | [OppTimeline](/simpler-grants-protocol/protocol/schemas/opp-timeline)           | Yes      | Key dates for the opportunity                                |
  | source         | [url](/simpler-grants-protocol/protocol/schemas/base-types#url)                 | No       | URL for the original source of the opportunity               |
  | customFields   | [Record](/simpler-grants-protocol/protocol/schemas/types/other#record)\<[CustomField](/simpler-grants-protocol/protocol/schemas/custom-field)\> | No       | Additional custom fields specific to this opportunity        |
  | createdAt      | [isoDateTime](/simpler-grants-protocol/protocol/schemas/base-types#isodatetime) | Yes      | The timestamp (in UTC) at which the record was created       |
  | lastModifiedAt | [isoDateTime](/simpler-grants-protocol/protocol/schemas/base-types#isodatetime) | Yes      | The timestamp (in UTC) at which the record was last modified |
  </TabItem>

  <TabItem label="TypeSpec">
  ```tsp
  @doc("A funding opportunity")
  model OpportunityBase {
    /** Globally unique id for the opportunity */
    @visibility(Lifecycle.Read)
    id: uuid;

    /** Title or name of the funding opportunity */
    title: string;

    /** Status of the opportunity */
    status: OppStatus;

    /** Description of the opportunity's purpose and scope */
    description: string;

    /** Details about the funding available */
    fundingDetails: FundingDetails;

    /** Key dates for the opportunity, such as when the application opens and closes */
    keyDates: OppTimeline;

    /** URL for the original source of the opportunity */
    source?: url;

    /** Additional custom fields specific to this opportunity */
    customFields?: Record<CustomField>;

    // Spreads the fields from the Metadata model into the Opportunity model
    ...SystemMetadata;
  }
  ```
  </TabItem>
