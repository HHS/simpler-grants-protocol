System-managed metadata that is automatically added to records.

### Data model

<Tabs>
  <TabItem label="Table">

| Property       | Type                                                                            | Required | Description                                                  |
| -------------- | ------------------------------------------------------------------------------- | -------- | ------------------------------------------------------------ |
| createdAt      | [utcDateTime](/simpler-grants-protocol/protocol/schemas/types/date#utcdatetime) | Yes      | The timestamp (in UTC) at which the record was created       |
| lastModifiedAt | [utcDateTime](/simpler-grants-protocol/protocol/schemas/types/date#utcdatetime) | Yes      | The timestamp (in UTC) at which the record was last modified |

  </TabItem>
