## Other numeric types

TypeSpec supports several other numeric types, including `float` and `decimal`. A full list of these types can be found in the [TypeSpec documentation](https://typespec.io/docs/language-basics/built-in-types/#numeric-types).

:::caution
Because these other numeric types have limited support in JSON schema, they are NOT RECOMMENDED for use within a CommonGrants API specification.
:::
