### boolean

A value that is either `true` or `false`, equivalent to `boolean` in JSON Schema.

<SchemaFormatTabs
  jsonSchema={{
    code: `
  $schema: https://json-schema.org/draft/2020-12/schema
  $id: boolean.yaml
  type: boolean
  description: A value that is either true or false
  `,
  }}
  example={{
    code: `true`,
  }}
/>

### Array

An ordered list of values, equivalent to `array` in JSON Schema.

#### Formats

<SchemaFormatTabs
  jsonSchema={{
    code: `
  $schema: https://json-schema.org/draft/2020-12/schema
  $id: array.yaml
  type: array
  description: An ordered list of values
  `,
  }}
  example={{
    code: `[1, 2, 3]`,
  }}
/>

#### Usage

In TypeSpec, you can specify the type of the values in the array with [templating](https://typespec.io/docs/language-basics/templates/):

```tsp
/** An array of integers */
alias IntegerArray = Array<integer>
```

This is equivalent to the following JSON Schema:

```yaml
$id: IntegerArray.yaml
type: array
items:
  type: integer
```

### Record

A collection of key-value pairs, equivalent to `object` in JSON Schema.

#### Formats

<SchemaFormatTabs
  jsonSchema={{
    code: `
  $schema: https://json-schema.org/draft/2020-12/schema
  $id: record.yaml
  type: object
  description: A collection of key-value pairs
  `,
  }}
  example={{
    code: `
  {
    "foo": 1,
    "bar": 2
  }`,
  }}
/>

#### Usage

In TypeSpec, you can specify the type of the values in the record with [templating](https://typespec.io/docs/language-basics/templates/):

```tsp
/** A record with a string key and an integer value */
alias StringIntRecord = Record<integer>
```

This is equivalent to the following JSON Schema:

```yaml
$id: StringIntRecord.yaml
type: object
additionalProperties:
  type: integer
```

If you want to define specific keys in a record, you can do so with a [`model`](https://typespec.io/docs/language-basics/models/).

### unknown

The `unknown` type accepts any value. It is equivalent to an empty JSON Schema.

<SchemaFormatTabs
  jsonSchema={{
    code: `
  $schema: https://json-schema.org/draft/2020-12/schema
  $id: unknown.yaml
  description: A value with any type
  `,
  }}
  example={{
    code: `
  // String
  "foo"

// Integer
123

// Boolean
true`,
}}
/>
