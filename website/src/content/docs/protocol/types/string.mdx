## employerTaxId

An Employer Identification Number (EIN).

<Tabs>
  <TabItem label="Example">

```json
"12-3456789"
```

  </TabItem>

  <TabItem label="JSON Schema">

```yaml
$schema: https://json-schema.org/draft/2020-12/schema
$id: employerTaxId.yaml
```
