### isoDate

A date on a calendar in ISO 8601 format YYYY-MM-DD.

<Tabs>
  <TabItem label="Example">

```json
"2025-01-01"
```

  </TabItem>

  <TabItem label="JSON Schema">

```yaml
$schema: https://json-schema.org/draft/2020-12/schema
$id: isoDate.yaml
type: string
format: date
description: A date on a calendar in ISO 8601 format YYYY-MM-DD
```

  </TabItem>

  <TabItem label="TypeSpec">

```tsp
/** A date on a calendar in ISO 8601 format YYYY-MM-DD */
@example(isoDate.fromISO("2025-01-01"))
scalar isoDate extends plainDate;
```

  </TabItem>
</Tabs>

### isoTime

A time on a clock, without a timezone, in ISO 8601 format HH:mm:ss.

:::caution
The `isoTime` type does not include timezone information. When representing a time in a specific timezone, use `utcDateTime` or `offsetDateTime` instead.
:::

<Tabs>
  <TabItem label="Example">

```json
"17:00:00"
```

  </TabItem>

  <TabItem label="JSON Schema">

```yaml
$schema: https://json-schema.org/draft/2020-12/schema
$id: isoTime.yaml
type: string
format: time
description: A time on a clock, without a timezone, in ISO 8601 format HH:mm:ss
```

  </TabItem>

  <TabItem label="TypeSpec">

```typespec
/** A time on a clock, without a timezone, in ISO 8601 format HH:mm:ss */
@example(isoTime.fromISO("17:00:00"))
scalar isoTime extends plainTime;
```

  </TabItem>
</Tabs>

### utcDateTime

A date and time with timezone in ISO 8601 format YYYY-MM-DDThh:mm:ssZ.

<Tabs>
  <TabItem label="Example">

```json
"2024-02-28T17:00:00Z"
```

  </TabItem>

  <TabItem label="JSON Schema">

```yaml
$schema: https://json-schema.org/draft/2020-12/schema
$id: utcDateTime.yaml
type: string
format: date-time
description: A date and time with timezone in ISO 8601 format YYYY-MM-DDThh:mm:ssZ
```

  </TabItem>
</Tabs>

### offsetDateTime

A date and time with timezone in ISO 8601 format YYYY-MM-DDThh:mm:ss±hh:mm.

<Tabs>
  <TabItem label="Example">

```json
"2024-02-28T17:00:00+01:00"
```

  </TabItem>

  <TabItem label="JSON Schema">

```yaml
$schema: https://json-schema.org/draft/2020-12/schema
$id: offsetDateTime.yaml
type: string
format: date-time
description: A date and time with timezone in ISO 8601 format YYYY-MM-DDThh:mm:ss±hh:mm
```

  </TabItem>
</Tabs>

## calendarYear

A calendar year in the format YYYY.

<Tabs>
  <TabItem label="Example">

```json
"2025"
```

  </TabItem>
  <TabItem label="JSON Schema">

```yaml
$schema: https://json-schema.org/draft/2020-12/schema
$id: calendarYear.yaml
type: string
pattern: ^[0-9]{4}$
description: A calendar year in the format YYYY
```

  </TabItem>
  <TabItem label="TypeSpec">

```typespec
/** A calendar year in the format YYYY */
@example("2025")
@pattern("^[0-9]{4}$")
scalar calendarYear extends string;
```

  </TabItem>
</Tabs>
