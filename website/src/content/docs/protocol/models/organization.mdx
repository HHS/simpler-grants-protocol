## Organization

An organization that can apply to or host funding opportunities.

### Table

| Property    | Type                                                                        | Required | Description                                                                  |
| ----------- | --------------------------------------------------------------------------- | -------- | ---------------------------------------------------------------------------- |
| id          | [uuid](/protocol/types/string#uuid)                                         | Yes      | The organization's unique identifier.                                        |
| name        | [string](/protocol/types/string#string)                                     | Yes      | The organization's legal name as registered with relevant authorities.       |
| address     | [Address](/protocol/fields/address)                                         | No       | The organization's primary mailing address.                                  |
| type        | [PCSOrgType](/protocol/fields/pcs#pcsorgtype)                               | No       | The organization's type within the Philanthropy Classification System (PCS). |
| ein         | [employerTaxId](/protocol/types/string#employertaxid)                       | No       | The organization's Employer Identification Number (EIN).                     |
| uei         | [samUEI](/protocol/types/string#samuei)                                     | No       | The organization's Unique Entity Identifier (UEI).                           |
| duns        | [duns](/protocol/types/string#duns)                                         | No       | The organization's Data Universal Numbering System (DUNS) number.            |
| mission     | [string](/protocol/types/string#string)                                     | No       | The organization's mission statement.                                        |
| yearFounded | [calendarYear](/protocol/types/date#calendaryear)                           | No       | The calendar year the organization was founded.                              |
| socials     | [Record](/protocol/types/other#record)\<[OrgSocialLinks](#orgsociallinks)\> | No       | The organization's social media and web presence links.                      |

model Organization {
    /** The organization's unique identifier. */
    id: Types.uuid;

    /** The organization's legal name as registered with relevant authorities. */
    name: string;

    /** The organization's primary mailing address. */
    address?: Fields.Address;

    /** The organization's type within the Philanthropy Classification System (PCS). */
    type?: Fields.PCSOrgType;

    /** The organization's Employer Identification Number (EIN), a unique identifier assigned by the IRS. */
    ein?: Types.employerTaxId;

    /** The organization's Unique Entity Identifier (UEI) from SAM.gov, used for federal contracting. */
    uei?: Types.samUEI;

    /** The organization's Data Universal Numbering System (DUNS) number, a unique identifier for businesses. */
    duns?: Types.duns;

    /** The organization's mission statement. */
    mission?: string;

    /** The calendar year the organization was founded. */
    yearFounded?: Types.calendarYear;

    /** The start date of the organization's fiscal year, used for financial reporting. */
    @example(plainDate.fromISO("2024-01-01"))
    fiscalYearStart?: plainDate;

    /** The end date of the organization's fiscal year, used for financial reporting. */
    fiscalYearEnd?: plainDate;

    /** Collection of the organization's social media and web presence links. */

}

## OrgSocialLinks

A record of the organization's social media and web presence links.

### Table

| Property     | Type                                                                        | Required | Description                                                          |
| ------------ | --------------------------------------------------------------------------- | -------- | -------------------------------------------------------------------- |
| website      | [url](/protocol/types/string#url)                                           | No       | The organization's primary website URL.                              |
| facebook     | [url](/protocol/types/string#url)                                           | No       | The organization's Facebook profile URL.                             |
| twitterOrX   | [url](/protocol/types/string#url)                                           | No       | The organization's Twitter/X profile URL.                            |
| bluesky      | [url](/protocol/types/string#url)                                           | No       | The organization's Bluesky profile URL.                              |
| instagram    | [url](/protocol/types/string#url)                                           | No       | The organization's Instagram profile URL.                            |
| linkedin     | [url](/protocol/types/string#url)                                           | No       | The organization's LinkedIn profile URL.                             |
| otherSocials | [Record](/protocol/types/other#record)\<[url](/protocol/types/string#url)\> | No       | Additional social media profiles not covered by the standard fields. |
