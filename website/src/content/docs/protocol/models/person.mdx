## Person

A person affiliated with an organization or grant application.

### Table

| Property  | Type                                                | Required | Description                  |
| --------- | --------------------------------------------------- | -------- | ---------------------------- |
| name      | [Name](/protocol/fields/name)                       | Yes      | The person's full name       |
| title     | [string](/protocol/types/string#string)             | No       | The person's title           |
| addresses | [PersonAddressCollection](#personaddresscollection) | No       | The person's addresses       |
| phones    | [PersonPhoneCollection](#personphonecollection)     | No       | The person's phone numbers   |
| fax       | [Phone](/protocol/fields/phone)                     | No       | The person's fax number      |
| emails    | [PersonEmailCollection](#personemailcollection)     | No       | The person's email addresses |
