Key dates in the opportunity's timeline, such as when the application opens and closes.

### Table

| Property    | Type                                                                      | Required | Description                                                                |
| ----------- | ------------------------------------------------------------------------- | -------- | -------------------------------------------------------------------------- |
| appOpens    | [Event](/protocol/fields/event)                                           | No       | The date (and time) at which the opportunity begins accepting applications |
| appDeadline | [Event](/protocol/fields/event)                                           | No       | The final deadline for submitting applications                             |
| otherDates  | [Record](/protocol/types/other#record)\<[Event](/protocol/fields/event)\> | No       | An optional map of other key dates in the opportunity timeline             |
