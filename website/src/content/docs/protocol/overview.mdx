## Library components

The `@common-grants/core` library is organized into the following namespaces. View the docs for each namespace to learn more about how to use them:

| Namespace                                                  | Description                                                              |
| ---------------------------------------------------------- | ------------------------------------------------------------------------ |
| [Types](/simpler-grants-protocol/protocol/types)           | Defines a base set of reusable data types                                |
| [Fields](/simpler-grants-protocol/protocol/fields)         | Defines a standard set of fields used to build data models               |
| [Models](/simpler-grants-protocol/protocol/models)         | Defines a collection of data models used within the protocol             |
| [Pagination](/simpler-grants-protocol/protocol/pagination) | Defines a set of parameters and patterns for paginated routes            |
| [Sorting](/simpler-grants-protocol/protocol/sorting)       | Defines a set of parameters and patterns for routes that support sorting |
| [Filters](/simpler-grants-protocol/protocol/filters)       | Defines a set of models for routes that support filtering                |
| [API](/simpler-grants-protocol/protocol/openapi)           | Defines the OpenAPI specification for the protocol                       |
