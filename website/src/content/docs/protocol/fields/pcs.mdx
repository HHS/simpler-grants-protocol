## PCSTerm

A term in the Philanthropy Classification System (PCS).

Visit [taxonomy.candid.org](https://taxonomy.candid.org/) for more information.

### Table

| Property       | Type   | Required | Description                                |
| -------------- | ------ | -------- | ------------------------------------------ |
| term           | string | Yes      | The term for the organization type.        |
| Classification | string | Yes      | The class for the organization type.       |
| code           | string | Yes      | The code for the organization type.        |
| description    | string | No       | The description for the organization type. |

## Sub-types

PCS terms are divided into the following sub-types based on the `class` property:

- [PCSOrgType](#pcsorgtype)
- [PCSSubject](#pcssubject)
- [PCSPopulation](#pcspopulation)
- [PCSSupportStrategy](#pcssupportstrategy)
- [PCSTransactionType](#pcstransactiontype)

### PCSOrgType

A [PCSTerm](#pcsterm) where the `class` is `"Organization types"`. Categorizes the type of organization receiving philanthropic support.

See [PCS organization types](https://taxonomy.candid.org/organization-type) for more information.

### PCSSubject

### PCSPopulation

A [PCSTerm](#pcsterm) where the `class` is `"Population groups"`. Categorizes the population being served by philanthropic activities.

See [PCS population groups](https://taxonomy.candid.org/population-group) for more information.

### PCSSupportStrategy

### PCSTransactionType

A [PCSTerm](#pcsterm) where the `class` is `"Transaction types"`. Categorizes the type of transaction used to support philanthropic activities. For example, "Grant" or "Loan".

See [PCS transaction types](https://taxonomy.candid.org/transaction-type) for more information.
