## EmailCollection

A collection of email addresses for a person or organization.

### Table

| Property    | Type                                                                            | Required | Description                                             |
| ----------- | ------------------------------------------------------------------------------- | -------- | ------------------------------------------------------- |
| primary     | [email](/protocol/types/string#email)                                           | Yes      | The primary email address for a person or organization  |
| otherEmails | [Record](/protocol/types/other#record)\<[email](/protocol/types/string#email)\> | No       | Additional email addresses keyed by a descriptive label |

/** A collection of email addresses. */
model EmailCollection {

  primary: Types.email;

  /** Additional email addresses keyed by a descriptive label (e.g., "work", "personal", "support"). */
  otherEmails?: Record<Types.email>;
}
