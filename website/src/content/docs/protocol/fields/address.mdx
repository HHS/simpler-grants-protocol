An address is a physical location.

### Table

| Property  | Type                                      | Required | Description                                                                  |
| --------- | ----------------------------------------- | -------- | ---------------------------------------------------------------------------- |
| Street 1  | [string](/protocol/types/string#string)   | Yes      | The primary street address line.                                             |
| Street 2  | [string](/protocol/types/string#string)   | No       | Additional street address information (e.g., apartment number, suite, etc.). |
| City      | [string](/protocol/types/string#string)   | Yes      | The city of the location.                                                    |
| State     | [string](/protocol/types/string#string)   | Yes      | The state of the location.                                                   |
| Zip       | [string](/protocol/types/string#string)   | Yes      | The postal code of the location.                                             |
| Latitude  | [number](/protocol/types/numeric#numeric) | Yes      | The latitude coordinate of the address location.                             |
| Longitude | [number](/protocol/types/numeric#numeric) | Yes      | The longitude coordinate of the address location.                            |
| Geography | [Record](/protocol/types/other#record)    | No       | Additional geospatial data in GeoJSON format.                                |

### Formats

## AddressCollection

A collection of addresses for a person or organization.

### Table

| Property       | Type                                                                          | Required | Description                                        |
| -------------- | ----------------------------------------------------------------------------- | -------- | -------------------------------------------------- |
| primary        | [Address](/protocol/fields/address)                                           | Yes      | The primary address for a person or organization   |
| otherAddresses | [Record](/protocol/types/other#record)\<[Address](/protocol/fields/address)\> | No       | Additional addresses keyed by a descriptive label. |

### Formats

<Tabs>
  <TabItem label="Example">

A JSON example of this model is:

```json
{
  "primary": {
    "street1": "123 Main St",
    "city": "Anytown",
    "stateOrProvince": "CA",
    "country": "US",
    "postalCode": "12345"
  },
  "otherAddresses": {
    "work": {
      "street1": "456 Main St",
      "street2": "Suite 100",
      "city": "Anytown",
      "stateOrProvince": "CA",
      "country": "US",
      "postalCode": "12345"
    },
    "international": {
      "street1": "123 Rue Principale",
      "city": "Montreal",
      "stateOrProvince": "QC",
      "country": "CA",
      "postalCode": "H2Y 1C6"
    }
  }
}
```

  </TabItem>

/** A collection of addresses. */
model AddressCollection {
  /** The primary address for a person or organization. */

  /** Additional addresses keyed by a descriptive label (e.g., "work", "home", "international"). */
  otherAddresses?: Record<Address>;
}
