A phone number.

### Table

| Property    | Type                                     | Required | Description                                                |
| ----------- | ---------------------------------------- | -------- | ---------------------------------------------------------- |
| countryCode | [string](/protocol/types/string#string)  | Yes      | The international country code (e.g., "+1" for US/Canada). |
| number      | [string](/protocol/types/string#string)  | Yes      | The local phone number without the country code.           |
| extension   | [string](/protocol/types/string#string)  | No       | Optional extension number for the phone line.              |
| isMobile    | [boolean](/protocol/types/other#boolean) | No       | Indicates whether this is a mobile/cell phone number.      |

The TypeSpec code for this model is:

```tsp
model Phone {
    /** The international country code (e.g., "+1" for US/Canada). */
    @pattern("^\\+[1-9][0-9]{1,3}$")
    countryCode: string;

    /** The local phone number without the country code. */
    number: string;

    /** Optional extension number for the phone line. */
    extension?: string;

    /** Indicates whether this is a mobile/cell phone number. */
    isMobile?: boolean = false;
}
```
