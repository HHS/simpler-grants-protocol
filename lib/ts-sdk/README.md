# @common-grants/sdk

The CommonGrants protocol TypeScript SDK.

## Installation

```bash
npm install @common-grants/sdk
```

## Development

```bash
# Install dependencies
npm install

# Build
npm run build

# Run tests
npm test

# Run tests with coverage
npm run test:coverage

# Lint
npm run lint

# Format
npm run format

# Run all checks
npm run checks
```
